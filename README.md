# Basic Booking Site
Consist of  **Booking Page** and **Ticket Page**

## Booking Page
### Destination input:
- Go From
- Go To
### Date Input:
- Departure Date
- Return Date
### Ticket Input:
- Three different ticket types
- Updates prices simultaneously

### Price Table
- Shows before tax price
- Shows tax
- Shows total (gross+tax)

## Ticket Page
Includes all input values from Booking page.
